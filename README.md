## Convention

### java Convention
- **directory** : 소문자로만 작성합니다.
  - ex) main, setting
- **ClassName** : 각 단어의 첫번째 문자는 대문자로 시작합니다.
  - ex) ScoreList
- **변수 및 함수명** : 첫 단어는 소문자, 두번째 단어부터는 대문자로 시작합니다. camelCase
  - ex) mainPanel

### commit Type
- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 수정
- style: 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- refactor: 코드 리팩토링
- test: 테스트 코드, 리팩토링 테스트 코드 추가
- chore: 빌드 업무 수정, 패키지 매니징
